Mechanical Vs. Membrane

Do mechanical keyboards pose an advantage of precision and speed over membrane keyboards?

When choosing a keyboard there are many different options available to the consumers. All these options are relating to the design of the keyboard whether that is the size of the keys, the spacing between each key, the height of the keyboard or the type of switches behind each key because of this there are a lot of things to consider before purchasing a keyboard.

Mechanical keyboards use actual, physical switches underneath the keys that determine when the user has pressed a key. Which does not sound like an extravagant design at all. When pressing on the key you can feel and hear the switch which provides greater tactile feedback. Compared with a membrane keyboard which uses a rubber switch to push through a hole in the middle membrane to connect the top and bottom membranes, which then creates an electrical circuit that sends the input of which key was pressed to the PC.

We asked ourselves after looking into both mechanical and membrane keyboards if one keyboard will pose a significant increase in accuracy and typing speed, mechanical keyboards marketing strategy was focused around providing the consumer with more accurately pressed keys and gaining a better typing speed because of this we want to prove mechanical keyboards purpose to be true on a bunch of different users from different backgrounds but with a very capable typing ability already.
